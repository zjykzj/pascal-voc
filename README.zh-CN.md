## 背景

[Pascal VOC](http://host.robots.ox.ac.uk/pascal/VOC/)是一个常用的轻量级目标识别和检测数据集。创建这个仓库的目的是为了更好地使用这个数据集，并将一些常用的实现和文档保存在这里。
